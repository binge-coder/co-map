# First time setup
- make venv :
```
python -m venv venv
```

- activate venv :
```
 ./venv/Scripts/Activate.ps1
```

- install requirements.txt :
```
pip install -r requirements.txt 
```

- run the app :
```
python app.py
```

- deactivate venv (optional) :
```
deactivate
```
